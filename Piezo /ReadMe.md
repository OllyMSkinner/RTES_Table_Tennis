## ADS1115 Pin connection list:
    * VDD - Pin 1
    * SDA - Pin 3
    * SCL - Pin 5
    * A0 - Pin 11
    * ADDR - GND
    * GND - GND
    * ALRT - positive piezo

## Piezo pin connection 
    * Negative - GND
    * Positive - ALRT
    * Both - connect to resistor
