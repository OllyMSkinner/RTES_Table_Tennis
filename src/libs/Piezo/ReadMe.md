## ADS1115 Pin Connection List

| ADC Connections | GPIO Pin | Physical Pin |
|---|---|---|
| VDD | 3.3v | 17 |
| SDA | GPIO 2 | 3 |
| SCL | GPIO 3 | 5 |
| A0 | Positive Piezo | N/A |
| ADDR | GND | 6 |
| GND | GND | 6 |
| ALRT | GPIO 26 | 37 |


## Piezo Pin Connection 

| Piezo | Connections |
|---|---|
| Positive | A0 |
| Negative | GND |
| Both | 330&Omega; Resistor |
